# Installation Attempt with ChatGPT Client

- **Command**
  - `npx -y install-mcp@latest https://api.supermemory.ai/mcp --client chatgpt --oauth=yes`
- **Result**
  - Installation aborted with `Invalid client: chatgpt` and a list of supported clients.
- **Notes**
  - The installer currently recognises clients such as `claude`, `cline`, `vscode`, and others; a `chatgpt` client is not available.
  - Supermemory cannot presently be installed for ChatGPT via this tool.
